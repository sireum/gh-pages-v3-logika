IntelliJ-based Integrated Verification Environment (IVE)
########################################################

To aid the development of Logika proofs, you can use
the Sireum IntelliJ plugin that provides an IVE for Logika.
Below is a screenshot of the IVE:

.. raw:: html

   <a href = "../_images/syntax-message.png"><img src="../_images/syntax-message.png" width="800"/></a>

Features
********

The IVE provides the following features:

* Syntax highlighting and error/warning reporting similar to most
  Integrated Development Environments (IDEs) for popular programming
  languages.

* Verification engine running in the background as you type.

* Can be used in conjunction with with IntelliJ's Scala Worksheet (``.sc`` files)
  or its Scala editor (``.scala`` files) to benefit from IntelliJ IDE features.
  The screenshot above illustrates how Logika IVE works with the Scala Worksheet.
