    2. IntelliJ-based Logika Integrated Verification Environment (LIVE)¶

    To aid the development of Logika proofs, you can use the Sireum IntelliJ plugin that includes LIVE. Here is a LIVE screenshot:

    2.1. Features¶

    LIVE provides the following features:

    • Syntax highlighting and error/warning reporting similar to most Integrated Development Environments (IDEs) for popular programming languages.
    • Convenient keyboard shortcuts for inserting logical symbols.
    • Verification engine running in the background to check proofs as you type.
    • Can be used in conjunction with IntelliJ’s Scala Worksheet (.sc files) or its Scala editor (.scala files) to benefit from IntelliJ IDE features. The screenshot above illustrates how LIVE works with the Scala Worksheet.

    2.2. LIVE Install¶

    The Sireum IntelliJ plugin that LIVE is a part of is currently under review for approval for distribution through JetBrains’ plugins website. For now, you can download the plugin zip file, and install it in IntelliJ. To do that, open IntelliJ’s Preferences (shortcut: ⌘, in OS X); then click on “Plugins” (1), select “Install from disk...” (2), select the downloaded plugin file, and restart IntelliJ.
